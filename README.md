This is a project to make a six-wheel robot with Arduino and Raspberry Pi.  
It consists with

- Six-wheel car platform
- Dual VNH3SP30 Motor Driver Carrier MD03A
- Arduino
- Raspberry Pi
- Wifi module
- camera
